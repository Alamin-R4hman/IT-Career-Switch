You: so what if an elemnt is a block elemnt and i set it as display inline

If you set a block element to display: inline;, it will behave like an inline element. This means:

Width and Height: The element will only take up as much width as its content requires. You cannot set its width and height. Line Behavior: The element will sit inline with other elements, not starting on a new line. Box Model: Margins and padding will still apply, but the element will not break to a new line.
