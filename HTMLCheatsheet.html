HTML Cheatsheet

Elements

Name Tag Description
Ordered List <ol> Creates ordered list items
Unordered List <ul> Creates unordered list items
List <li> Creates list items within the <ul> or <ol> tags
Video <video src=""> Embeds a media player. Src will contain URL to video. Adding 'controls' will give control to video
Emphasis <em> Emphasises text. Browsers will usually italicise the text
Div <div> Div standing for division is basically a container for specific info you want in the container.
Line Break <br> Will craete a line break
Image <image src=""> Embed images in documents. The src attribute contains the image url and is mandatory
Heading <h1> - <h6> Different levels of heading ranging between 1 - 6 (1 being the biggest)
Paragraph <p> Creates a paragraph element
Body <body> Defines the body of the document
Span <span> Can be used to style or customise code within the tags
Strong <strong> Makes text within it bold
